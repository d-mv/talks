Be(st|tter) practicies

While styling has been most of the time a matter of preference, nowdays certain ways of doing it have been followed:

Be(st|tter) practicies

  • Prefer individual, local style -> .Component__button_title
  • Avoid using tag selector -> a, p, div & etc
  • Use pre-processor (for faster development) and post-processor (to automate production, compatibility features & etc)
  • Avoid overlaping classes
  • If there is a chance -> stick to cheaper styles
  • Avoid using JS where it's not need to reduce load on JS browser engine

Flex vs Grid

Both concepts are intended for layout of elements. Flex is a purely responsive concept. The flexibility of grid is that it can provide both fixed and responsive layout

Grid is also the shortest way to center elements:

{ display: grid; place-items: center; }
